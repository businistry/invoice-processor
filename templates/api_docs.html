API Documentation

The Invoice Pro API is available on Professional and Enterprise plans. It allows you to programmatically process invoices and integrate with your existing systems.
Authentication

All API requests require authentication using an API key. You can generate an API key in the Settings page.

Example Request
curl -X POST https://api.invoice-pro.com/v1/process \
-H "Authorization: Bearer YOUR_API_KEY" \
-H "Content-Type: application/json" \
-d '{"options": {"hotel_code": "STLMO", "auto_detect": true}}'
Endpoints

Process invoices and extract information.

Request Parameters
Parameter Type Required Description
file File Yes The PDF invoice file to process
options.hotel_code String No Default hotel code to use (e.g., "STLMO")
options.auto_detect Boolean No Whether to auto-detect hotel location (default: true)
options.model String No AI model to use (default: "gpt-4o")

{
  "success": true,
  "data": {
    "vendor": "AcmeCorp",

}

Code Examples
import requests

API_KEY = "your_api_key"
API_URL = "https://api.invoice-pro.com/v1"

def process_invoice(file_path, hotel_code="STLMO", auto_detect=True):
    headers = {
        "Authorization": f"Bearer {API_KEY}"
    }
    
    files = {
        "file": open(file_path, "rb")
    }
    
    data = {
        "options.hotel_code": hotel_code,
        "options.auto_detect": "true" if auto_detect else "false"
    }
    
    response = requests.post(f"{API_URL}/process", headers=headers, files=files, data=data)
    return response.json()

# Example usage
result = process_invoice("path/to/invoice.pdf")
print(result)
// Using fetch API
const API_KEY = 'your_api_key';
const API_URL = 'https://api.invoice-pro.com/v1';

async function processInvoice(fileInput, options = {}) {
  const formData = new FormData();
  formData.append('file', fileInput.files[0]);
  
  // Add options
  if (options.hotelCode) {
    formData.append('options.hotel_code', options.hotelCode);
  }
  
  formData.append('options.auto_detect', options.autoDetect !== false);
  
  const response = await fetch(`${API_URL}/process`, {
    method: 'POST',
    headers: {
      'Authorization': `Bearer ${API_KEY}`
    },
    body: formData
  });
  
  return response.json();
}

// Example usage
document.getElementById('invoiceForm').addEventListener('submit', async (e) => {
  e.preventDefault();
  const fileInput = document.getElementById('invoiceFile');
  const result = await processInvoice(fileInput, {
    hotelCode: 'STLMO',
    autoDetect: true
  });
  console.log(result);
});
